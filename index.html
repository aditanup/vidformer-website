Enabling instant visualization, efficient loading, fast processing, and remote access for video data.

VidFormer is not a Lightroom-style video editor, video database, or computer vision library.

*Although, it is highly complementary to these.

VidFormer is a declarative video editor for data workloads.

VidFormer is an efficient video processing execution engine.

VidFormer is an infrastructure-grade video result streaming server.

*And you can get started here.

Easy Access

Store videos in various different locations - cloud, disk, server, etc. - and interact with them like they are on your own laptop. VidFormer supports dozens of storage protocols.

About

VidFormer is an open-source library created by Dominik Winecki, PhD Student in Computer Science at The Ohio State University, Columbus. It is spun out of the Interactive Data Systems lab led by Prof. Arnab Nandi.

VidFormer is a continuation of the research presented in V2V (ICDE 2024) that introduces a system to efficiently synthesize video results for video queries. V2V returns a fully-edited video, enabling the user to consume results in the same manner as the source videos.

V2V introduced a grammar to express video transformations in a declarative manner and a heuristic optimizer that improves the efficiency of V2V processing in a manner similar to how databases execute relational queries. Experiments show that the V2V optimizer enables video synthesis to run 3x faster.

VidFormer is an Rust-based extension of the system presented in V2V, and provides a suite of tools that enables enables people to create videos without worrying about performance or efficiency. VidFormer can be used for a variety of use-cases, including:

  • Creating videos as results to queries in video search engines (e.g., "show me every time zebras fight, and overlay the time, date, location, and animal IDs").
  • Visualization of computer vision models (e.g., Drawing bounding over objects in videos in a python notebook).
  • Video data cleaning and preprocessing (i.e. filtering, sampling, scaling, etc.) for computer vision model training.

However, VidFormer is not limited to these use-cases. Ultimately, it is a universally applicable tool that enables instant visualization, efficient loading, and remote access for video data.
